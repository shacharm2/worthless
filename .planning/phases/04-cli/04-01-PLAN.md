    - ensure_home() is idempotent (second call is a no-op)
    - WorthlessHome dataclass provides .db_path, .fernet_key, .shard_a_dir, .lock_file properties
    - rewrite_env_key(env_path, var_name, new_value) atomically replaces only the target key's value
    - rewrite_env_key preserves comments, blank lines, ordering, other variables
    - rewrite_env_key with nonexistent var_name raises KeyError
    - scan_env_keys(env_path) returns list of (var_name, value, provider) for detected API keys
    - scan_env_keys skips values with Shannon entropy < 4.5 (placeholders like "sk-your-key-here")
    - scan_files(paths, enrollment_data) returns list of ScanFinding objects
    - ScanFinding has: file, line, var_name, provider, is_protected (True if matches known decoy)
    - scan_files with decoy data marks matching values as is_protected=True
    - Shannon entropy("sk-your-key-here") < 4.5 (skipped)
    - Shannon entropy of a real 48-char random key > 4.5 (detected)
    - .lock-in-progress file creation uses O_CREAT|O_EXCL for atomicity
  </behavior>
  <action>
    1. Create `src/worthless/cli/bootstrap.py`:
       - WorthlessHome dataclass with base_dir: Path (default ~/.worthless/)
       - Properties: db_path -> base_dir/worthless.db, fernet_key_path -> base_dir/fernet.key, shard_a_dir -> base_dir/shard_a, lock_file -> base_dir/.lock-in-progress
       - fernet_key property: reads fernet.key file, returns bytes
       - `ensure_home(base_dir=None) -> WorthlessHome`: Create dirs (0700), generate Fernet key if missing (write with 0600), initialize DB via ShardRepository.initialize(). Idempotent.
       - `acquire_lock(home: WorthlessHome) -> contextmanager`: Creates .lock-in-progress with O_CREAT|O_EXCL, deletes in finally. Raises WorthlessError(LOCK_IN_PROGRESS) if exists.
       - `check_stale_lock(home: WorthlessHome)`: If .lock-in-progress exists, check if it's stale (>5 min old), remove if stale, raise if fresh.

    2. Create `src/worthless/cli/dotenv_rewriter.py`:
       - `shannon_entropy(s: str) -> float`: Standard Shannon entropy calculation
       - `scan_env_keys(env_path: Path) -> list[tuple[str, str, str]]`: Read .env, find lines matching KEY_PATTERN, filter by entropy > 4.5, return (var_name, value, provider) tuples
       - `rewrite_env_key(env_path: Path, var_name: str, new_value: str) -> None`: Line-by-line replacement preserving all other content. Atomic write via tempfile + os.replace(). Raise KeyError if var_name not found.

    3. Create `src/worthless/cli/scanner.py`:
       - ScanFinding dataclass: file (str), line (int), var_name (str | None), provider (str), is_protected (bool), value_preview (str, fully masked by default)
       - `load_enrollment_data(home: WorthlessHome | None) -> set[str]`: Read shard_a files to get known decoy values. Return empty set if no home dir (CI mode).
       - `scan_files(paths: list[Path], enrollment_data: set[str] | None = None) -> list[ScanFinding]`: Scan each file line-by-line for KEY_PATTERN matches. Check entropy > 4.5. If enrollment_data provided and value matches, mark is_protected=True. Return all findings.
       - `format_sarif(findings: list[ScanFinding], tool_version: str) -> dict`: SARIF v2.1.0 output per research spec.

       - tests/test_scanner.py: pattern detection, entropy thresholding, decoy suppression, SARIF format validation, file scanning
