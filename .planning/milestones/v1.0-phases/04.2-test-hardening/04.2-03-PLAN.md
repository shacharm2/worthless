- Install uv, `uv sync --extra dev --extra test`

- Run: `uv run ruff check .`

- `if: github.event_name == 'pull_request'`
- `needs: test`
- Uses `py-cov-action/python-coverage-comment-action@v3`
- Permissions: `pull-requests: write`

- `continue-on-error: true`
