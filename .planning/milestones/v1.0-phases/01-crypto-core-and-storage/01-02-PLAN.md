  <behavior>
    - ShardRepository.store(alias, shard_b, commitment, nonce, provider) encrypts shard_b with Fernet before INSERT
    - ShardRepository.retrieve(alias) decrypts shard_b_enc with Fernet, returns (shard_b, commitment, nonce, provider) or None
    - Raw SQLite column shard_b_enc != plaintext shard_b
    - ShardRepository.list_keys() returns list of enrolled aliases
    - Metadata persists across close/reopen of the database connection
    - Duplicate alias raises IntegrityError or similar
  </behavior>
  <action>
1. Create `src/worthless/storage/repository.py` with class `ShardRepository`:
   - `__init__(self, db_path: str, fernet_key: bytes)`: stores db_path, creates `Fernet(fernet_key)` instance
   - `async def initialize(self) -> None`: calls `init_db(self.db_path)` from schema module
   - `async def store(self, alias: str, shard_b: bytes, commitment: bytes, nonce: bytes, provider: str) -> None`:
     - Encrypt: `shard_b_enc = self._fernet.encrypt(shard_b)`
     - INSERT into shards table. Let SQLite raise on duplicate alias.
   - `async def retrieve(self, alias: str) -> tuple[bytes, bytes, bytes, str] | None`:
     - SELECT from shards WHERE key_alias = alias
     - If no row, return None
     - Decrypt: `shard_b = self._fernet.decrypt(row["shard_b_enc"])`
     - Return (shard_b, commitment, nonce, provider)
   - `async def list_keys(self) -> list[str]`:
     - SELECT key_alias FROM shards, return list
   - `async def set_metadata(self, key: str, value: str) -> None`:
     - INSERT OR REPLACE into metadata table
   - `async def get_metadata(self, key: str) -> str | None`:
     - SELECT from metadata WHERE key = key
   - Use `aiosqlite.connect(self.db_path)` as async context manager in each method (simple approach for PoC; connection pooling is not needed)
   - Set `row_factory = aiosqlite.Row` for dict-like access

2. Update `src/worthless/storage/__init__.py` to export: `ShardRepository`, `init_db`
