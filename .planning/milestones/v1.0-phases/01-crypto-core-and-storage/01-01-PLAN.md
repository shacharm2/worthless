<task type="auto" tdd="true">
  <name>Task 2: Implement crypto primitives to make tests pass</name>
  <files>src/worthless/crypto/splitter.py, src/worthless/crypto/__init__.py</files>
  <behavior>
    - split_key(b"sk-test") returns SplitResult where shard_a XOR shard_b == original key
    - split_key produces shards of same length as input
    - Neither shard equals the original key (probabilistic but virtually certain for non-trivial keys)
    - reconstruct_key with valid shards returns bytearray of original key
    - reconstruct_key with tampered shard raises ShardTamperedError
    - secure_key context manager zeros bytearray on exit
    - split_key(b"") raises ValueError
  </behavior>
  <action>
1. Create `src/worthless/crypto/splitter.py` with:
   - `split_key(api_key: bytes) -> SplitResult`:
     - Validate input: raise `ValueError` if empty
     - Generate mask with `secrets.token_bytes(len(api_key))`
     - `shard_a = bytes(a ^ b for a, b in zip(api_key, mask))`
     - `shard_b = mask`
     - Generate nonce with `secrets.token_bytes(32)`
     - `commitment = hmac.new(nonce, api_key, hashlib.sha256).digest()`
     - Return `SplitResult(shard_a, shard_b, commitment, nonce)`
   - `reconstruct_key(shard_a: bytes, shard_b: bytes, commitment: bytes, nonce: bytes) -> bytearray`:
     - `key = bytearray(a ^ b for a, b in zip(shard_a, shard_b))`
     - `expected = hmac.new(nonce, bytes(key), hashlib.sha256).digest()`
     - If `not hmac.compare_digest(expected, commitment)`: zero key, raise `ShardTamperedError`
     - Return key
   - `secure_key(key_buf: bytearray)` context manager:
     - `yield key_buf` in try block
     - In finally: `key_buf[:] = b'\x00' * len(key_buf)`

2. Update `src/worthless/crypto/__init__.py` to export: `split_key`, `reconstruct_key`, `secure_key`, `SplitResult`, `ShardTamperedError`
