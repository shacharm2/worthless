## Decisions Made
- **Anti-enumeration (H-2/M-3):** Unknown endpoints return uniform 401 instead of 404 to prevent endpoint discovery. Updated existing tests that expected 404.
- **Upstream sanitization (M-4):** Parse upstream JSON error, preserve error type, replace message with generic "upstream provider error".
- **relay_response aread():** For non-SSE responses sent via stream=True, call `await response.aread()` before accessing `.content`.
- **Tasks 1+2 merged:** The streaming, gate ordering, zeroing, and error handling changes are tightly coupled (same request handler), so they were implemented as a single TDD cycle.
