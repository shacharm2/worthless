Worthless makes leaked keys worthless. How?

Worthless replaces each protected API key in your .env with a share that cannot call the provider alone. Your app still works through a local proxy that holds the other share. See how it works.

What Worthless does not protect against

  • Compromised machine.
  • Provider compromised.
  • Early build limits.

Worthless is not a sandbox and not the only layer you need. Use it with a vault, secret scanning, and key rotation. Full threat model: security model.

Scanners find leaks. Vaults store secrets. Provider dashboards and gateways manage spend. Worthless makes a copied .env AI-key value unusable on its own.
