Vulnerability reporting for Worthless

Worthless is open-source software provided AS IS under AGPL-3.0. This page tells you where to send security reports. It creates no obligation, no warranty, and no service-level commitment.

Reporting a vulnerability

If you found a security issue, please do not open a public GitHub issue. Use one of these private channels:

  • GitHub private vulnerability reporting: github.com/oblangatas/worthless/security/advisories/new
  • Email: security@wless.io

Useful things to include if you can: the Worthless version or commit SHA, your operating system, and a reproduction case (steps, code, or proof-of-concept). Encrypted email is welcome; mention it in your first message.

No commitments

Worthless is maintained by a single person as unpaid open-source work. Response to security reports is best effort only.

  • No guaranteed response time, and no SLA.
  • No obligation to triage, fix, or respond to any report.
  • The maintainer may be unavailable for arbitrary periods (travel, day job, vacation, illness, life). That is normal and expected for solo open-source.

AGPL-3.0 sections 15 and 16 disclaim all warranties and limit liability. Nothing on this page modifies that, and nothing here grants you any rights you would not otherwise have. If you need guaranteed response times, security warranties, or indemnification, a commercial license is what that is for: legal@wless.io.

Coordinated disclosure, a request

We prefer that researchers give the project a chance to fix issues before public disclosure. We cannot require this. You may disclose on whatever timeline you choose. A common-sense window is appreciated where the issue is not actively exploited.

Good-faith research

The maintainer does not intend to pursue legal action against good-faith research that tests only against your own installation, does not access data that does not belong to you, and does not run scanners against shared infrastructure without coordination. This is a statement of intent. It is not a legal safe-harbor agreement.
