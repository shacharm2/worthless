why it matters

Worm-shaped package compromise turns every trusted maintainer into a possible distribution path. That is why token scope and rotation matter more than vibes.

What gets stolen here is not only one secret. The dangerous part is publishing power: npm tokens, GitHub tokens, and other credentials that let the attacker act as someone trusted.

how it got there

The attack moved through the package ecosystem. Once a trusted account or package path is abused, downstream users can receive malicious code as if it came from the normal maintainer.

worthless angle

For supported locked AI keys, public copied material is a worse credential. For npm tokens, GitHub tokens, and broad cloud access, Worthless is not the control.
