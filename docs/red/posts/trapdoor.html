TrapDoor hid in npm, PyPI, and crates.

Different ecosystems, same old door. A package gets installed, code runs, and secrets nearby become interesting.

Source: The Hacker News

why it matters

Cross-ecosystem campaigns are a reminder that this is not an npm story. Python, Rust, JavaScript: developers install things, and install paths are powerful.

The stolen material can be anything the process can read: tokens, environment values, local config, or cloud credentials. The consequence is not one clean failure. It depends on what that developer or build runner was trusted to do.

how it got there

The campaign spread through public package registries. That puts the attack inside a normal developer habit: adding dependencies.
