# Adversarial Review

Date: 2026-03-30

This directory contains the static adversarial review package for the current
Worthless Python PoC.

Files:

- `attack-map.md` — attacker types, motivations, and the highest-risk attack chains
- `operator-hardening.md` — practical deployment guidance and non-negotiables
- `redteam-checklist.md` — manual validation scenarios before stronger claims ship
- `security-claims.md` — wording guidance so product claims match the actual boundary

Recommended reading order:

1. `attack-map.md`
2. `operator-hardening.md`
3. `redteam-checklist.md`
4. `security-claims.md`
